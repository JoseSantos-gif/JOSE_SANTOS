📘 Rotación 3D

Vértices del Cubo

Representan los puntos (x, y, z) del cubo.
Se mantienen proporcionales para evitar deformación.

Aristas

Definen qué vértices se conectan.
Permiten dibujar la estructura del cubo.

Rotación en X

Se aplica la transformación:

y' = y·cos(a) - z·sin(a)
z' = y·sin(a) + z·cos(a)

Rotación en Y

Se aplica la transformación:

x' = x·cos(a) + z·sin(a)
z' = -x·sin(a) + z·cos(a)

Proyección 3D → 2D

Convierte puntos 3D en coordenadas de pantalla.

factor = d / (d + z)
Genera efecto de perspectiva.

Dibujo del Cubo

moveTo() + lineTo(): dibujan las aristas.
stroke(): muestra el contorno.

Animación

angleX / angleY: controlan la rotación.

Se incrementan en cada frame para animación continua.
