📘 Escalado 3D

Vértices del Cubo

Se definen como coordenadas 3D (x, y, z).
Representan las esquinas del cubo.

Aristas

Son pares de índices que conectan vértices.
Permiten dibujar la estructura del cubo.

Escalado 3D

Se multiplica cada coordenada por un factor:

x' = x · scale
y' = y · scale
z' = z · scale

Esto cambia el tamaño del objeto.

Proyección 3D → 2D

Convierte coordenadas 3D a pantalla 2D.

Usa perspectiva:
factor = 200 / (z + distancia)

Dibujo del Cubo

moveTo() + lineTo(): dibujan cada arista.
stroke(): muestra el contorno.

Animación de Escalado

scale: aumenta o disminuye el tamaño.
growing: controla si crece o reduce.

Animación Continua

requestAnimationFrame(draw):
Actualiza el cubo en cada frame.
