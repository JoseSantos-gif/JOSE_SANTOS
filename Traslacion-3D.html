📘 Traslación 3D con Límites

Vértices del Cubo

Definen la forma del objeto en 3D (x, y, z).
En este caso es un cubo pequeño centrado en el origen.

Aristas

Conectan los vértices para formar el cubo.
Se dibujan mediante líneas.

Traslación

Se aplica sumando desplazamientos:

x' = x + tx
y' = y + ty
z' = z + tz

Permite mover el objeto en el espacio.

Velocidad

vx, vy, vz:
Controlan la velocidad de movimiento en cada eje.

Límites (Colisión)

Se define un rango (limit).

Cuando el cubo lo supera:
v *= -1

Esto invierte la dirección (rebote).

Rotación

Se aplica sobre los ejes X y Y usando funciones trigonométricas.
Hace que el cubo gire mientras se mueve.

Proyección

Convierte coordenadas 3D a 2D.
En este caso es una proyección simple (ortográfica).

Animación

requestAnimationFrame(draw):
Ejecuta el render en cada frame para movimiento continuo.
