📘 Sistema Vectorial

Canvas y Contexto

getContext("2d"):
Permite dibujar gráficos en el canvas.

Coordenadas del Mouse

mousemove: detecta el movimiento del cursor.

clientX / clientY: obtienen la posición del mouse en pantalla.

Centro del Sistema

Se define el origen del vector en el centro del canvas:

centroX = width / 2
centroY = height / 2

Definición de Vector

Un vector se representa como una línea entre dos puntos:

moveTo(inicio)
lineTo(fin)

Dibujo del Vector

beginPath(): inicia el trazo
stroke(): dibuja la línea
strokeStyle: color del vector
lineWidth: grosor de línea

Animación

requestAnimationFrame(dibujar):
Permite actualizar el vector en tiempo real según el mouse.
