🏀 Malla Poligonal 3D con Three.js

Librería Three.js

La línea:

<script src="https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.min.js"></script>

importa la librería Three.js desde internet mediante un CDN.

Three.js permite crear gráficos y animaciones 3D usando WebGL directamente en el navegador.

Creación de la Escena

const scene = new THREE.Scene();

Crea el espacio tridimensional donde se colocarán todos los objetos, cámaras y luces.

Color de Fondo

scene.background = new THREE.Color(0x111111);

Define el color del fondo de la escena.

El código hexadecimal 0x111111 representa un tono gris oscuro.

Cámara Perspectiva

THREE.PerspectiveCamera() crea una cámara 3D similar a la visión humana.

Parámetros:
• 75 → Campo de visión (FOV)
• aspect → Relación ancho/alto
• 0.1 → Distancia mínima visible
• 1000 → Distancia máxima visible

Posición de la Cámara

camera.position.z = 4;

Aleja la cámara 4 unidades en el eje Z para poder visualizar correctamente la esfera.

Renderizador WebGL

const renderer = new THREE.WebGLRenderer();

El renderizador dibuja todos los gráficos 3D usando la GPU mediante WebGL.

Tamaño del Renderizador

renderer.setSize(window.innerWidth / 2, window.innerHeight);

Configura el tamaño del área 3D.

• window.innerWidth / 2 → mitad de pantalla
• window.innerHeight → altura completa

Inserción del Canvas

document.getElementById("contenedor3D") .appendChild(renderer.domElement);

Inserta el canvas generado por Three.js dentro del contenedor HTML.

Geometría Esférica

THREE.SphereGeometry() crea una esfera formada por polígonos.

Parámetros:
• 1.5 → radio
• 16 → segmentos horizontales
• 16 → segmentos verticales

Material de la Esfera

THREE.MeshBasicMaterial() define la apariencia visual del objeto.

Configuración:
• color: 0x00ffcc
• wireframe: true

Efecto Wireframe

wireframe: true

Permite visualizar únicamente las líneas de la malla poligonal, mostrando la estructura geométrica.

Creación del Mesh

const sphere = new THREE.Mesh( geometry, material );

Combina la geometría y el material para formar el objeto 3D visible.

Agregar Objeto a la Escena

scene.add(sphere);

Inserta la esfera dentro de la escena tridimensional.

Función de Animación

function animate()

Esta función se ejecuta continuamente para generar movimiento en tiempo real.

requestAnimationFrame

requestAnimationFrame(animate)

Ejecuta la animación de forma fluida sincronizada con los FPS del navegador.

Rotación de la Esfera

sphere.rotation.x += 0.01;
sphere.rotation.y += 0.01;

Genera la rotación continua sobre los ejes X e Y.

Renderizado de la Escena

renderer.render(scene, camera);

Dibuja la escena completa desde la perspectiva de la cámara.

Inicio de la Animación

animate();

Inicia el ciclo infinito de animación de la esfera.

Evento Resize

window.addEventListener("resize", () => {})

Detecta automáticamente cuando cambia el tamaño de la ventana del navegador.

Actualización del Aspect Ratio

camera.aspect = (window.innerWidth / 2) / window.innerHeight;

Ajusta la proporción de la cámara para evitar deformaciones.

Actualización de Proyección

camera.updateProjectionMatrix();

Actualiza internamente la proyección de la cámara.

Redimensionamiento del Renderizador

renderer.setSize( window.innerWidth / 2, window.innerHeight );

Cambia el tamaño del renderizador para mantener el diseño responsive.
